# Traffic Shaping lab description
This lab shows how to control the traffic between hosts that are communicating. The network scenario is composed of just three devices: two hosts and a router. To shape the traffic exchanged between the two computers you just have to add the command in the startup file of the router.
To summarize all the types of traffic control you can do, inside the r.startup file I wrote all the commands needed to complete the task. To execute a specific command, you just have to erase the comment symbol of the line you need.
